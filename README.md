# tou-tv-xmbc

From code.google.com/p/tou-tv-xmbc

This project is a XBMC plugin used to stream video from the French Canadian web site Tou.tv.

You can follow the installation manual from the xbmc.org wiki for information about how to install plugins. You will probably want use the manual installation at the bottom of the page. http://wiki.xbmc.org/?title=HOW-TO_install_and_use_plugins_in_XBMC
